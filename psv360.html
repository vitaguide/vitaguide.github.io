Part I: Requirements

  • A PS Vita running firmware 3.60
  • The latest release of HENkaku Offline Installer (The VPK File).
  • The latest release of VitaShell (The VPK File).
  • An active Wi-Fi connection.
  • An FTP client such as FileZilla.

Part II: Running and Installing HENkaku

  1. Open the PS Vita's Web Browser and go to http://henkaku.xyz.
  2. Click the orange 'Install' button.
  3. Click the next orange 'Install button' (This will run the exploit and install molecularShell).
    • Installation can take anywhere from 15 seconds to a minute.
    • If any error message appears or your console freezes, see the troubleshooting page.
  4. When the Installation of HENkaku and molecularShell is finished, you will be brought back to the LiveArea screen.

Part III: Installing VitaShell & Henkaku Offline Installer

  1. Open molecularShell.
  2. Press Start to open the HENkaku Settings.
  3. Navigate to 'Enable unsafe homebrew' and press X.
    • WARNING: WHEN INSTALLING VPKs BE CAREFUL! A MISUSED, MISCONFIGURED, OR MALICIOUS VPK CAN PERMANENTLY DAMAGE YOUR DEVICE!
  4. Dismiss the warning by pressing X.
  5. Exit the HENkaku settings menu by pressing Circle.
  6. Press the PS button to go back to the LiveArea screen and close molecularShell.
  7. Relaunch molecularShell.
  8. Press Select to start the FTP Server.
  9. Connect to your Vita using your FTP client and navigate to 'ux0:'.
  10. Copy the 'offlineinstaller.vpk' file to ux0:.
  11. Copy the 'VitaShell.vpk' file that you downloaded earlier to ux0:.
  12. Press Circle with the FTP server dialog open to close the FTP Server.
  13. Navigate to ux0: within molecularShell.
  14. Select 'VitaShell.vpk' and press X.
  15. Press X to install VitaShell.
  16. Press X at the warning box.
  17. Select 'offlineinstaller.vpk' and press X.
  18. Press X to install offlineinstaller.
  19. Press X at the warning box.
  20. Exit molecularShell


Part IIII: Installing Henkaku Offline Exploit

  1. Open 'offlineinstaller'.
  2. Press X to install the exploit.
  3. Once the exploit is done installing, press any button to exit.
  4. It is recommended to test the exploit after installing.
  5. Power Off your console.
  6. Wait about 5 seconds after the PS button stops blinking.
  7. Turn your console back on.
  8. Enable Airplane mode.
  9. Open the Email app.
  10. If you already had an email account attached to your Console, Choose HENkaku Offline (If you didn't, skip this step.)
  11. Select Inbox
  12. Select the email, and wait 10-15 seconds.
    • If you get an error, Close the Email app and try again.
  13. If the exploit was successful, you should be back on the LiveArea screen.
    • Now you can use this exploit to launch HENkaku without an internet connection!
