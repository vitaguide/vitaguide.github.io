How to set your PSN account data aka 'Initial Signin' 
1. Set Henkaku DNS in your Connection Settings as Primary DNS if not done yet here the IP for the Primary DNS :    // OR use vitaupdateblocker
2. Install registryeditor
3. Navigate to /config/np/ and make the keys for your email and password blank
4. Install uricaller
6. Open uricaller and call the uri "psnreg:"
7. If you have HenkakuDNS or Vitaupdateblocker running the app will just work
8. Sign into or Create your PSN Account
9.Press done and the app will auto-exit and assign your account to the system
