Step 2.5:

Manual Update to 3.60

The HENkaku exploit only runs on firmware version 3.60. If you are on a firmware version below 3.60, You will need to manually update to 3.60.

Part I: HENkaku Update Server

  1. Open the Settings app.
  2. Select 'Network'.
  3. Select 'Wi-Fi Settings'
  4. If you are not connected to the internet, do so now.
  5. Select your current connection. (Should have a green dot to the left.)
  6. Select 'Advanced Settings'
  7. Under 'DNS Settings', choose Manual
